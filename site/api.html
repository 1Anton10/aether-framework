CLI

  • create — scaffold приложения
  • dev / start — HMR, SSR, DSM
  • build / deploy — production
  • migrate — Next / Nuxt / Vue / Svelte / Angular / Solid / Qwik / Lit / Vite → Aether

HTTP / DSM

  • POST /api/delta — [slot u32][len u32][payload…]
  • POST /api/effect — effect → resume; заголовок X-Aether-Effect-Mode
  • GET /api/catalog — JSON каталога (demo host)
  • WS /aether-dsm · /api/dsm — тот же кодек
  • GET /app.wasm · /app.gc.wasm · /aether.program.json
  • GET /api/ssr — stream HTML

Handlers

  • inc / dec / add — локальный mutate
  • set — LocalSet
  • server: true — DSM mutate через /api/delta
  • perform — effect → слот into
  • effect types: catalog, set, add, mul, rtt

Runtime

  • AetherRuntime — mount / hydrate
  • AetherRouter — client router
  • Wasm exports: aether_init, apply_delta, handlers

Packages

  • aether · aether/ssr · aether/router
  • aether_ssr — renderToString / stream
  • vite-plugin-aether
  • create-aether · aether_cli
  • aether-compat-react|vue|svelte|solid|qwik|lit|angular — алиасы при migrate

Синтаксис → IR

Парсеры: JSX/TSX, Vue SFC, Svelte, Angular template, Solid, Qwik, Lit, HTML. Это компиляция шаблона в IR, не полный runtime исходного фреймворка. Nuxt / Next / SvelteKit — через migrate + парсер базового синтаксиса.
