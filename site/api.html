CLI

  • create <name> — scaffold приложения
  • dev / start — HMR + SSR + DSM
  • build / deploy — production bundle
  • migrate — Next / Nuxt / Vite → Aether + src/pages

HTTP / DSM

  • POST /api/delta — binary deltas [slot u32][len u32][payload…]
  • POST /api/effect — effect frame → resume delta
  • WS /aether-dsm · WS /api/dsm — same codec; effect prefix 0xEF
  • GET /app.wasm · /app.gc.wasm · /aether.program.json

Bindings handlers

  • { "op": "inc"|"dec"|"add", "slot", "delta?" }
  • { "op": "set", "slot", "delta": value } — absolute LocalSet
  • { "op": "inc", "slot", "server": true } — DSM server mutate
  • { "op": "perform", "effect", "into" } — algebraic effect

Runtime (browser)

  • AetherRuntime — mount / hydrate SSR (data-aether-nid)
  • window.AetherRouter — createRouter, bindLinks
  • Exports: aether_init, apply_delta, aether_resume, handlers

Packages

  • aether / aether/ssr / aether/router
  • aether_ssr — renderToString, renderToStream
  • vite-plugin-aether — virtual aether:program
  • create-aether — scaffold
  • aether-compat-* — React / Vue / Solid / Qwik / Lit / Angular
