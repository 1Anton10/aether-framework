Frontend framework

Aether

Compile UI to Wasm. Resume from memory. Patch the DOM. Sync over binary DSM.

Один пайплайн для JSX, Vue, Svelte, Angular, Solid, Qwik, Lit и HTML. Состояние живёт в Wasm-памяти — без Virtual DOM и без JSON-RPC.

Get Started

Как у Vite / Nuxt: создал проект → npm run dev → открыл браузер.

  1. Создайте проект 
    node <aether-framework>/packages/aether_cli/src/index.ts create my-app
cd my-app
  2. Соберите компилятор (один раз в framework) 
    cd <aether-framework>
cargo build -p aether_compiler
npm run build -w aether_runtime
  3. Запустите dev-сервер 
    npm run dev
# → http://localhost:5173

Что внутри

Полный цикл: compile → resume → patch → sync.

Wasm memory

Linear + WasmGC heap. Snapshot в HTML, cold start без гидратации.

Incremental DAG

Dirty-queue и derived-слоты. Патчится только то, что изменилось.

Binary DSM

WebTransport / WebSocket / HTTP. Один бинарный кодек дельт.

Effects

Algebraic perform → server handler → resume в слот.

Any frontend

JSX · Vue · Svelte · Angular · Solid · Qwik · Lit · HTML → один IR.

Migrate

aether migrate подхватывает Vite, Next, Nuxt, Angular и ставит aliases.
