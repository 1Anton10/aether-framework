Guide

Пайплайн разработчика от нуля до sync.

1. Create

Сгенерируйте приложение (как npm create vue@latest / nuxi init):

node packages/aether_cli/src/index.ts create my-app
cd my-app
npm run dev

2. Edit UI

Пишите на любом поддерживаемом синтаксисе. Entry задаётся в aether.config.json.

export default function App() {
  return (
    <div>
      <h1>{count}</h1>
      <button onClick={inc_count}>+</button>
    </div>
  );
}

3. State & bindings

src/state.ts — начальные значения. aether.bindings.json — derived и handlers.

{
  "derived": [{ "target": "doubled", "from": "count", "op": "mul", "arg": 2 }],
  "handlers": {
    "inc_count": { "op": "inc", "slot": "count" }
  }
}

4. Dev server

  • / — документация / лендинг проекта
  • /app — скомпилированное приложение
  • HMR при изменении entry
  • DSM: /aether-dsm, /api/delta

5. Migrate

В существующем Vite / Next / Nuxt / Angular / Solid проекте:

node path/to/aether_cli create --migrate
# или: npm run migrate  (из monorepo)

6. Build

npm run build
# → dist_production / outDir с app.wasm + runtime
