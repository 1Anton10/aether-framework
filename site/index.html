Aether

UI → IR → Wasm memory → DOM patch → binary DSM

Фреймворк компилирует интерфейс в WebAssembly. Состояние живёт в памяти модуля, обновления идут точечными патчами по dirty DAG, синхронизация с сервером — бинарными дельтами. SSR отдаёт HTML с node id для гидратации.
