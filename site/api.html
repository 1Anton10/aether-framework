API

Конфиг, CLI и runtime-поверхность.

aether.config.json

{
  "entry": "src/App.tsx",
  "state": "src/state.ts",
  "bindings": "aether.bindings.json",
  "outDir": "dist",
  "wasmgc": true,
  "server": { "port": 5173 },
  "site": { "landing": "public/index.html", "publicDir": "public" },
  "routes": {
    "/": { "type": "static", "file": "public/index.html" },
    "/app": { "type": "app" }
  },
  "effects": {
    "db.get": { "type": "add", "arg": 100 }
  }
}

CLI

  • create <name> — scaffold приложения
  • start / dev — HMR + DSM
  • build — production bundle
  • deploy — bundle + cloud manifest
  • migrate — импорт из чужого стека

Handlers (bindings)

  • inc_<slot> / { "op": "inc", "slot" }
  • server_inc_<slot> — mutate через DSM
  • perform_<effect>__into_<slot> — algebraic effect

Runtime exports

  • aether_init / apply_delta / aether_resume
  • app.wasm + app.gc.wasm
  • snapshot: <script type="aether/snapshot">

Frontends

Entry может быть .tsx, .vue, .svelte, .html, Angular template, Solid, Qwik, Lit.
